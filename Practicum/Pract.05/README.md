# Теоретични задачи

**Въпрос 1:** Какво ще се изведе след изпълнение на кода?

```c++
#include <iostream>

void func(int& a, int& b) {
    ++a = b;
    b = a--;
}

int main() {
    int a = 5;
    int b = 10;
    func(a, b);
    std::cout << a << " " << b << std::endl;
}
```

**Въпрос 2:** Какво ще се изведе след изпълнение на кода?

```c++
#include <iostream> 

int& tricky(int& a) { 
	a++; 
	return a; 
}

int main() { 
	int x = 10;
	int& y = tricky(x);
	int z = tricky(x); 
	y++; 
	std::cout << x << " " << y << " " << z; 
}
```
