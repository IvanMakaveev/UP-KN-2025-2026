## Теоретични задачи

**Въпрос 1:** Какво ще се изведе на конзолата?

```c++
#include <iostream>
using namespace std;

int main()
{
    int number = 2;
    
    if (number < 2)
        cout << "first";
    else if (number = 0)
        cout << "second";
    else if (number >= 2)
        cout << "third";
    else
        cout << "unknown";
}
```
